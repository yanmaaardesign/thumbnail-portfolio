Admin Login

ログインして作品を管理

管理者だけがログインして、作品データ・ピックアップ表示・分類情報を更新するための画面です。

  • ピックアップ作品は `featured` で制御
  • 分類は 制作種別 / ジャンル の2軸で管理
  • ドラフト作品は公開サイトに表示しません

ログイン

Supabase Auth に登録した管理者アカウントでログインします。

作品一覧

公開状態と分類を一覧で確認できます。

Supabase 接続前はサンプル表示です。

新規作品を追加

新しい作品を登録するフォームです。既存作品の編集は一覧カードの「編集」から行えます。
